# Visitor Counter API

Simple visitor counter api

## API Reference

#### Get visitors

```
  GET /visitors/${key}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `key`     | `string` | **Required**. Your key |

#### Register key

```
  POST /register/${key}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `key`     | `string` | **Required**. Your key  |

#### Increment key

```
  POST /increment/${key}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `key`     | `string` | **Required**. Your key |

#### Reset key

```
  POST /reset/${key}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `key`     | `string` | **Required**. Your key |
